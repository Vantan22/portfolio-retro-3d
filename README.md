# Book Library Project

## Cài đặt

```bash
npm install framer-motion three gs
```

## Tính năng

- Hiệu ứng mở/đóng sách với Framer Motion
- Render 3D scene với Three.js
- Animations mượt mà với GSAP
- Âm thanh tương tác với Howler.js

## Cấu trúc màu
